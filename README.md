# Inventario Sublikami (OSBK)

Aplicación Android privada de control de inventario, visual minimalista y profesional para la marca Sublikami.

## 🚀 Características

- Dashboard intuitivo con tarjetas por producto
- Registro de ventas y actualización automática de stock
- Visualización clara en dispositivos móviles
- Integración con Firebase (agregar `google-services.json` en `/app`)
- Genera un APK listo para instalar y compartir

## 📦 Instalación y compilación

1. **Clona el repositorio:**
   
   ```
   git clone https://github.com/panditayt82-code/sbk82.git
   ```
2. Abre el proyecto en Android Studio

3. Agrega tu archivo `google-services.json` en la carpeta `app/` para habilitar Firebase (descárgalo desde la consola de Firebase).

4. Sincroniza y compila el proyecto

    - **Build > Build APK(s)**
    - El APK está en `/app/build/outputs/apk/release/`

5. Instala y comparte el APK:
    - Sube el archivo APK a Google Drive, Telegram o a tu propio sitio web
    - Descarga e instala en tu celular Android (activar “fuentes desconocidas”)

## 📝 Descripción corta para compartir la app

Inventario Sublikami es una aplicación privada para llevar control total de tu inventario, registrar ventas y visualizar existencias en tiempo real, desde el celular.
